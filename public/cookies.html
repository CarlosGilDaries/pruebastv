      Política de Cookies y Almacenamiento Local

      Última actualización: 13/08/2025

      Esta aplicación utiliza cookies, almacenamiento local del navegador (localStorage) y almacenamiento de sesión (sessionStorage) para su correcto funcionamiento.

      Todos los elementos que se almacenan son estrictamente necesarios para prestar el servicio y garantizar la seguridad de la plataforma, por lo que no es posible acceder ni utilizar la aplicación sin ellos.

      1. Responsable del tratamiento

      • Responsable: [Nombre o razón social]
      • Contacto: [Correo electrónico de contacto]
      • Base legal: Ejecución de un contrato (art. 6.1.b RGPD), ya que estos datos son imprescindibles para el uso del servicio solicitado por el usuario.

      2. Tipos de almacenamiento y datos utilizados

      a) Cookies

      • XSRF-TOKEN → Protege contra accesos no autorizados mediante falsificación de peticiones.
      • disfrutavalenciatv_session → Mantiene información esencial para la navegación entre páginas.

      Duración: válidas solo durante la sesión del navegador.

      b) Almacenamiento Local (localStorage)

      Estos datos permanecen en el dispositivo incluso después de cerrar el navegador o apagar el equipo, para que el usuario no tenga que iniciar sesión cada vez que accede.

      • auth_token → Token de autenticación enviado por el backend para validar el acceso.
      • current_user_email → Email del usuario actual, usado para comprobación adicional junto con device_id.
      • device_id_[email] → Identificador único del dispositivo asociado a la cuenta (formato device_id_ + current_user_email para permitir múltiples cuentas en el mismo dispositivo).

      Duración: hasta que el usuario los elimine manualmente desde la configuración del navegador.

      c) Almacenamiento de Sesión (sessionStorage)

      Estos datos se eliminan automáticamente al cerrar el navegador.

      • plan_id → Identificador del plan a adquirir, usado en el proceso de pago (PayPal o Redsys).
      • months → Duración del plan contratado (3 o 12 meses).
      • actual_plan → Plan activo del usuario, para verificar el acceso a contenidos.
      • needed_plans → Planes requeridos si el contenido solicitado necesita una suscripción superior.
      • movie_id → Identificador de un recurso de pago por visión o alquiler para procesar el pago.

      3. Derechos del usuario

      El usuario puede acceder, rectificar o suprimir sus datos, así como ejercer otros derechos reconocidos por el RGPD, escribiendo desde nuestra página de Contacto.

      También puede eliminar manualmente los datos almacenados en su dispositivo desde las opciones de su navegador.

      4. Más información

      Para más detalles sobre el tratamiento de datos personales, consulte nuestra Política de Privacidad.
